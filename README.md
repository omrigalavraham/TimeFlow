## 🌟 למה TimeFlow?

בעולם של הסחות דעת, TimeFlow היא המקום השקט שלך. בנינו מערכת שמשלבת **ניהול משימות**, **טכניקות פוקוס**, ו**בקרת התקדמות** בממשק אחד נקי ומינימליסטי. זה לא עוד Todo List - זה כלי לניהול החיים שלך.

### ✨ פיצ'רים מרכזיים

#### 🍅 **מצבי פוקוס מתקדמים**
*   **פומודורו:** סשנים ממוקדים של 25 דקות.
*   **Deep Work:** עבודה עמוקה של 90 דקות ללא הפרעות.
*   **Blitz Mode:** ספרינטים מהירים של 15 דקות לחיסול משימות קטנות.
*   *עיצוב מרהיב שמשתנה לפי המצב כדי להכניס אותך לזון.*

#### 🎓 **Smart Study Hub (קמפוס דיגיטלי)**
*   מעקב אחרי מטלות אקדמיות, קורסים ומבחנים.
*   **חישוב עומס אוטומטי:** המערכת תגיד לך מתי אתה עמוס מדי.
*   **מצב למידה:** טיימרים מותאמים ללמידה עם הפסקות מתוזמנות.

#### 🧠 **Strategic Calendar**
*   מבט על השבוע שלך עם Drag & Drop מלא.
*   **תכנון יומי חכם:** אשף תכנון שעוזר לך לבחור מה לעשות היום לפי הזמן הפנוי שלך.
*   אינטגרציה מלאה בין המשימות ליומן.

#### 📱 **חווית מובייל ואפליקציה (PWA)**
*   התקן על האייפון/אנדרואיד שלך ישירות מהדפדפן.
*   ממשק מותאם לחלוטין למגע (Touch-first design).
*   תמיכה במצב כהה (Dark Mode) אוטומטי.

## 🛠️ טכנולוגיות

נבנה עם הסטאק המודרני והמהיר ביותר כדי להבטיח ביצועים וחווית משתמש חלקה:

*   **Framework:** [Next.js 14](https://nextjs.org/) (App Directory)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/) + [Framer Motion](https://www.framer.com/motion/) (Animations)
*   **State Management:** [Zustand](https://github.com/pmndrs/zustand)
*   **Database:** [Supabase](https://supabase.com/)
*   **Drag & Drop:** [dnd-kit](https://dndkit.com/)

## 🚀 איך להתחיל?

רוצה להריץ את זה אצלך?

1.  **שכפל את הפרויקט:**
    ```bash
    git clone https://github.com/omrigalavraham/TimeFlow.git
    cd TimeFlow
    ```

2.  **התקן תלויות:**
    ```bash
    npm install
    ```

3.  **הגדר משתני סביבה:**
    צור קובץ `.env.local` והוסף את מפתחות ה-Supabase שלך:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_url_here
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_key_here
    ```

4.  **הרץ את השרת:**
    ```bash
    npm run dev
    ```

פתחו את הדפדפן בכתובת `http://localhost:3000` ותתחילו לזרום!
